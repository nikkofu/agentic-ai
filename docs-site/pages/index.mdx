# Welcome to Agentic-AI

Agentic-AI is a high-performance, observable, and resilient multi-agent orchestration runtime kernel.

## Core Pillars
- **Persistence**: Built-in SQLite support via Prisma.
- **Extensibility**: Lightweight Skill-based plugin system.
- **Observability**: Real-time dashboard and OpenTelemetry tracing.
- **Resilience**: Sophisticated retry and model fallback logic.
